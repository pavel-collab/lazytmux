# LazyTmux

TUI-приложение для удобного управления tmux-сессиями, окнами и конфигурацией, написанное на Go с использованием [Bubble Tea](https://github.com/charmbracelet/bubbletea).

## Возможности

- Просмотр всех tmux-сессий, окон и панелей
- Создание, удаление и переименование сессий и окон
- Подключение к сессиям и отключение клиентов
- Разделение окон (горизонтально и вертикально)
- Редактор конфигурации tmux с GUI
- Управление плагинами (TPM)
- Навигация в стиле vim (hjkl)
- Двуязычный интерфейс (EN/RU)

## Требования

- Go 1.21+
- tmux 3.0+

## Установка

### Из исходного кода

```bash
# Клонировать репозиторий
git clone https://github.com/yourusername/lazytmux
cd lazytmux

# Собрать
make build

# Запустить
./bin/lazytmux
```

### Установка в систему

```bash
# Установить в $GOPATH/bin
make install

# После этого команда lazytmux доступна глобально
lazytmux
```

### Через go install

```bash
go install ./cmd/lazytmux
```

| `j` / `↓` | Вниз |
| `k` / `↑` | Вверх |
| `h` / `←` | Предыдущая панель |
| `l` / `→` | Следующая панель |
| `Tab` | Переключить панель |
| `Shift+Tab` | Предыдущая панель |

### Действия

| Клавиша | Действие |
|---------|----------|
| `n` | Создать сессию/окно |
| `d` | Удалить сессию/окно |
| `r` | Переименовать |
| `a` | Подключиться к сессии |

## Редактор конфигурации

LazyTmux включает встроенный редактор для файла `~/.tmux.conf`. Открывается клавишей `c`.

### Вкладки редактора

1. **Options** - настройки tmux (mouse, history-limit, status-bar и др.)
2. **Plugins** - управление плагинами через TPM

### Управление в редакторе

| Клавиша | Действие |
|---------|----------|
| `Tab` | Переключить вкладку (Options/Plugins) |
| `h/l` | Переключение между категориями и опциями |
| `j/k` | Навигация по списку |
| `Enter` | Редактировать значение |
| `Space` | Переключить значение (для boolean/выбора) |
| `L` | Переключить язык (EN/RU) |
| `s` | Сохранить изменения |
| `Esc` | Вернуться назад |

### Работа с плагинами

#### Предварительные требования

Перед работой с плагинами необходимо установить [Tmux Plugin Manager (TPM)](https://github.com/tmux-plugins/tpm):

```bash
# Установить TPM
git clone https://github.com/tmux-plugins/tpm ~/.tmux/plugins/tpm
```

#### Включение плагинов

1. Откройте редактор конфигурации (`c`)
2. Перейдите на вкладку Plugins (`Tab`)
3. Выберите плагин и нажмите `Space` для включения
4. Сохраните конфигурацию (`s`)
5. Перезагрузите tmux или нажмите `prefix + I` для установки плагинов

#### Доступные плагины

| Плагин | Описание |
|--------|----------|
| TPM | Менеджер плагинов (обязателен) |
| Sensible | Разумные настройки по умолчанию |
| Resurrect | Сохранение/восстановление сессий |
| Continuum | Автоматическое сохранение |
| Yank | Копирование в системный буфер |
| Logging | Логирование и захват экрана |
| Copycat | Поиск по regex |
| Open | Открытие файлов и URL |
| Pain Control | Управление панелями |
| Sessionist | Управление сессиями |
| Prefix Highlight | Индикатор prefix |
| CPU | Информация о CPU/RAM |
| Battery | Индикатор батареи |

## Тестирование

```bash
make test

# Сгенерировать HTML-отчет о покрытии
```

### Структура тестов

- `internal/tmux/*_test.go` - unit-тесты парсеров и типов

- `internal/ui/*_test.go` - тесты UI-компонентов

### Архитектура

Приложение построено на [Bubble Tea](https://github.com/charmbracelet/bubbletea) с использованием паттерна Elm Architecture:

1. **Model** - состояние приложения
2. **Update** - обработка событий, возврат нового состояния
3. **View** - рендеринг состояния в строку

#### Пакеты

- **internal/tmux** - взаимодействие с tmux через `os/exec`
- **internal/config** - чтение/запись `~/.tmux.conf`, определения опций и плагинов
- **internal/ui** - TUI на Bubble Tea, включая редактор конфигурации

### Команды Make

```bash
make help         # Показать все команды
make build        # Собрать бинарник
make test         # Запустить тесты

make clean        # Очистить артефакты сборки
```
